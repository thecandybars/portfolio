About me

Web developer and audio engineer based in Bogotá, Colombia

Hi, my name is Andrés Vilá. I ́m 45 years old and have worked almost all my life as an audio engineer and an educator.

Since I was a kid I ́ve been curious for programming languages and all this time I orbited around programming in different ways : at 12 years old I was coding my first text-based video games using Basic; I worked on my college years as a database programmer for my father ́s company; my undergraduate thesis (I studied Film) was a multimedia cd-rom fictional story; my masters degree (audio engineering) thesis was a software midi controller (it translated computer vision information into midi messages); the last 8 years I developed a personal brand of music synthesizers called “Sintetizadores para todos!” (Synthesizers for everybody!) creating hardware electronic musical instruments both analog and digital (using Arduino boards).

Projects

Some of my projects

Meme Factory is a simple app made in React for memes design.
It allows the user to select an image from a variety of classic memes, add and edit text properties, and download the final image.

NFTickets is an events marketplace based on a blockchain that delivers the tickets as NFTs.
Users can explore, search and select among musical, theatre and sport events, and even upload it´s own event to the blockchain.

World Wide Countries is a SPA coded with React that displays basic information about world countries.
Consumes data from a public API to build it´s own database.

Iceberg synth is a digital sound synthesizer built upon an Arduino using the Mozzi library.
The oscillator is an original design, capable of producing digital noise, fully tunable by the user.
Its sound can be shaped by a state variable filter and an envelope-controlled amplifier.
